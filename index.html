User Manual

Memory Functions Explained

  • MC (Memory Clear) - Erases the currently stored memory value back to 0.
  • MR (Memory Recall) - Replaces your current display with the stored memory value.
  • MS (Memory Store) - Stores the current displayed number into memory, overwriting anything previously saved.
  • M+ (Memory Add) - Adds the currently displayed number to the value already stored in memory.
  • M- (Memory Subtract) - Subtracts the currently displayed number from the value stored in memory.
