**Calc OS** is a powerful, highly customizable, all-in-one calculator application built specifically for Android. It goes far beyond standard math, offering a complete suite of tools including multi-mode calculators, 15+ converters, global time, a dictionary, and an integrated AI Math Assistant. 

Wrapped in a beautiful **Material Design 3 (Material You)** interface, it allows users to deeply personalize their experience with dynamic color palettes, custom Google Fonts, and adjustable button shapes.

## ✨ Key Features

### 🔢 Calculators
*   **Standard:** Clean, everyday calculator with memory functions (MC, MR, M+, M-, MS) and infinite history.
*   **Scientific:** Advanced functions (Trig, Log, Root, Powers) with Degree/Radian toggles.
*   **Programmer:** Hex, Dec, Oct, and Bin conversions with bitwise operations (AND, OR, XOR, LSH, RSH, MOD).

### 🔄 Converters & Tools
*   **15+ Categories:** Length, Weight, Volume, Area, Temp, Data, Speed, Time, Angle, Pressure, Energy, Power, Force, and Data Rate.
*   **Live Currency & Crypto:** Fetches live, up-to-date conversion rates for fiat (USD, EUR, GBP, etc.) and top cryptocurrencies (BTC, ETH, SOL).
*   **Financial:** Loan & Mortgage calculator estimating monthly payments and total interest.
*   **Health:** Built-in BMI calculator with Metric/Imperial support.
*   **Date Calc:** Accurately calculate the exact difference between two dates.

### 🧠 AI Math Assistant
*   **Context-Aware AI:** Powered by an LLM that remembers your last 20 messages.
*   **Flawless Math Rendering:** Automatically renders complex equations beautifully using **KaTeX** and Markdown.
*   **Modern UI:** Bubble-free, professional chat interface featuring a custom DiceBear avatar.

### 🌍 Smart APIs Integrated
*   **Dictionary:** Built-in glossary for quick mathematical definitions and general word lookups.
*   **World Time:** Look up accurate global dates, times, and timezone abbreviations instantly.

### 🎨 Deep Personalization (Material You)
*   **Dynamic Colors:** Pick a single HEX base color and the app's custom engine generates a full 13-color Material 3 palette natively.
*   **Typography:** Import any Google Font via URL. Save fonts as "pills" and apply them globally, or specifically to the AI Chat or Calculator displays.
*   **Button Shapes:** Switch calculator buttons between Pill (Circular), Rounded Rectangle (Squircle), or Square.
*   **Dark/Light Mode:** Full system-level themes.

## 🚀 Tech Stack

Originally built as an ultra-optimized single-file web application and converted into a native Android app via Webview wrapper.
*   **Frontend:** Vanilla JavaScript, HTML5, Tailwind CSS
*   **Icons:** Lucide Icons
*   **Math Rendering:** KaTeX & Marked.js
*   **APIs:** WorldTimeAPI, DictionaryAPI, Fawazahmed0 Currency API, PicoApps LLM

## 📦 Installation (Android)

1. Go to the [Releases](../../releases) tab.
2. Download the latest `CalcOS.apk` file.
3. Transfer it to your Android device (or download it directly on your phone).
4. Tap the APK to install (you may need to allow "Install from Unknown Sources" in your Android settings).
5. Open **Calc OS** and enjoy!
